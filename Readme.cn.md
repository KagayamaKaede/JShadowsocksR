# JShadowsocksR

[English version introduction](https://github.com/KagayamaKaede/JShadowsocksR/blob/master/Readme.cn.md)

这是一个用Java编写的 ShadowsocksR 库。
兼容 Shadowsocks 协议，并会在未来支持 ShadowsocksR 协议。

这个项目也许不会长期维护。

##编译准备

* JDK 1.7+

请使用最新版本的 Intellij IDEA 以减少编译时遇到问题的可能。
